# KPI
You can use it only inside private VPN and only after authorization.

Depends:
```
python-qt4
python3-pyqt4
```
Optional:
```
python-qt4-dev
pyqt4-dev-tools
```
Debian package can be found in branch with the same name. More information: https://github.com/ink-ru/KPI/blob/debian/opt/kpi/README.md
